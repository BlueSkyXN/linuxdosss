# Linux.do 论坛刷帖助手

一款面向 Linux.do 论坛的自动化辅助工具，提供图形界面，可自动浏览、点赞、回帖并跟踪信任等级进度。

## 功能概览

- **自动登录检测**：浏览器内手动登录，程序自动识别登录状态
- **等级/进度追踪**：读取当前等级、下一级要求，并在界面中实时更新
- **多板块随机浏览**：按勾选板块随机抽取帖子浏览
- **模拟阅读**：随机滚动+随机等待，模拟真实阅读行为
- **随机点赞与回帖**：可配置点赞率/回复率，包含回复模板
- **运行统计**：统计本次浏览、点赞、回复数量
- **系统托盘**：最小化到托盘、托盘菜单控制、悬停显示统计

## 快速开始

### 方式一：下载可执行文件

从 GitHub Releases 下载对应平台的可执行文件后直接运行即可。

### 方式二：从源码运行

**环境要求**

- Python 3.8+
- Chrome 浏览器

**安装依赖**

```bash
pip install -r requirements.txt
```

**运行**

```bash
python linux_do_gui.py
```

## 使用流程

1. 启动程序，按需填写代理（不需要可留空）
2. 勾选需要浏览的板块
3. 设置点赞率、回复率、等待时间
4. 点击“开始”，程序会打开浏览器
5. 在浏览器内完成登录，程序会自动检测并继续

> 登录检测会定期轮询，若长时间未登录会自动结束本次任务。

## 参数说明

| 参数 | 说明 |
|------|------|
| 代理 | HTTP 代理地址，例如 `127.0.0.1:7897`，留空则不使用代理 |
| 点赞率 | 对主帖点赞的概率（百分比） |
| 回复率 | 对帖子回复的概率（百分比） |
| 等待时间 | 操作间随机等待范围（秒），格式示例：`1-3` |
| 板块选择 | 勾选需要参与浏览的板块 |
